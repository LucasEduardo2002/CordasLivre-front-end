# Branding assets (CordasLivre)

Coloque aqui os arquivos finais da marca com estes nomes:

- `logo-full.png` (logo completa horizontal para header/hero)
  - recomendado: 1600x500 px
- `logo-mark.png` (símbolo circular)
  - recomendado: 1024x1024 px
- `logo-mark-180.png` (Apple touch icon)
- `logo-mark-192.png` (PWA icon)
- `logo-mark-512.png` (PWA icon)

## Regras para boa nitidez

- Exporte em PNG com fundo transparente.
- Gere a partir do arquivo vetorial original (SVG/AI/PDF), sem ampliar imagem pequena.
- Evite compressão agressiva para não serrilhar as linhas finas.
- Mantenha margem de respiro no símbolo para uso em favicon.
